# Lock In — Personal Ops Dashboard

A gamified career-progress tracker. XP, levels, streaks, and a **momentum meter that decays when you coast** — built to fight the exact pattern of getting comfortable and going quiet for weeks.

Tracks: a daily "one thing," cert/skill progress, concrete automation **Receipts** to show your manager, **Visibility** actions, and a **90-day plan to senior**. All state is saved in your browser's `localStorage` — your data never leaves your device.

## Run it

It's a single self-contained file. Just open `index.html` in any browser, or visit the GitHub Pages URL.

## Deploy on GitHub Pages

Settings → Pages → Build and deployment → Source: **Deploy from a branch** → Branch: `main` / `root`. Your dashboard goes live at `https://<username>.github.io/<repo>/`.

## Make it yours

`index.html` is a compiled bundle — edit the source design and re-export rather than hand-editing it.
